# Python - Data Structures

## Description
This project contains Python programs that demonstrate the use of lists, tuples, strings, and basic data structure operations.

## Learning Objectives
- Print and manipulate lists
- Access and replace list elements
- Work with copies of lists
- Remove characters from strings
- Print matrices
- Use tuples
- Return multiple values
- Find the maximum value in a list
- Check for multiples of 2
- Delete elements from a list

## Requirements
- Ubuntu 20.04 LTS
- Python 3.8.5
- All files start with:
  ```
  #!/usr/bin/python3
  ```
- Code follows `pycodestyle` (version 2.7)
- All files are executable

## Files

| File | Description |
|------|-------------|
| 0-print_list_integer.py | Prints all integers in a list |
| 1-element_at.py | Retrieves an element from a list |
| 2-replace_in_list.py | Replaces an element in a list |
| 3-print_reversed_list_integer.py | Prints a list in reverse |
| 4-new_in_list.py | Replaces an element in a copied list |
| 5-no_c.py | Removes 'c' and 'C' from a string |
| 6-print_matrix_integer.py | Prints a matrix of integers |
| 7-add_tuple.py | Adds two tuples |
| 8-multiple_returns.py | Returns string length and first character |
| 9-max_integer.py | Finds the largest integer in a list |
| 10-divisible_by_2.py | Finds multiples of 2 |
| 11-delete_at.py | Deletes an element at a given position |
| 12-switch.py | Switches the values of two variables
